nodejs中的URL处理方法

发表于 2017-10-30

今天的主角~~> 统一资源定位符，URL。

URL的几种处理数据的方法：

  • url.parse( )
  • url.format( )
  • url.resolve( )

先来说说

url.parse( urlStr,[parseQueryString],[slashesDenoteHost])；

打开node.exe，没下载的赶紧去官网下载。好了，开始。
第一个参数为url地址。
